== Description ==

= Ingles: =

Latest version of this fantastic plugin to support online through chat integrated into wordpress.

Some of its features are:
* 1. Multisession users
* 2. Choice of notification sounds
* 3. Enabling and Disabling chat
* 4. Easy integration for different languages
* 5. Configuration is not required for the installation of the chat. It installs itself and displayed in any theme.
* 6. Save all conversations, until the administrator decides to remove them.
* 7. Nice design and is easily modified.

= Espa&ntilde;ol: =

Ultima version de este fantastico plugin para dar soporte online mediante un chat integrado a wordpress.

Algunas de sus funcionalidades son:
* 1. Multisesion de usuarios
* 2. Eleccion de sonidos de notificaciones
* 3. Habilitacion y deshabilitacion del chat
* 4. Facil integracion para diferentes idiomas
* 5. No es necesaria ninguna configuracion para la instalacion del chat. Se instala solo y se muestra en cualquier theme.
* 6. Guarda todas las conversaciones, hasta que el administrador decida eliminarlas.
* 7. Diseño agradable y es de facil modificacion.

== Installation ==

= 1 = 

Descargas chatlive y subes a la carpeta /wp-content/plugins los archivos  descomprimidos, luego, instalas el plugin desde la seccion plugins de wordpress. si tu theme esta correctamente desarrollado debes ver el chat en el tu sitio, sin ningun tipo de errores, en caso contrario, revisa el archivo header.php del theme que estas utilizando y verfica si se realiza la llamada a la funcion wp_head(); justo antes del cierre de la etiqueta &lt;head&gt;.

Chatlive downloads and upload to the / wp-content/plugins unzipped files, then install the plugin from the plugins section of wordpress. if your theme is properly developed should see the chat on your site, without any kind of errors, otherwise, check the header.php file of the theme you're using and verfica if the call to the function wp_head (); just before the close of the &lt;head&gt;.

= 2. =

Recuerda que debes configurar los datos del chat, desde la seccion configuracion del chat.

You must configure the chat data from the configuration of the chat section.
